Passos para abrir no celular:
  1. Abra este arquivo no Chrome (Android) ou Safari (iPhone), não no visualizador do app.
  2. No Android: toque no cadeado/ícone de site → Permissões → Câmera: Permitir.
  3. No iPhone: Ajustes → Safari → Câmera (Perguntar/Permitir). No site: AA → Configurações do Site → Câmera: Permitir.
  4. Sem HTTPS, a câmera embutida pode não abrir. Use Foto (fallback).
